# IBM Cybersecurity SkillsBuild Portfolio

Bienvenido a mi portfolio de aprendizaje del certificado de ciberseguridad de IBM SkillsBuild.

Este sitio documenta mi progreso a través de las diferentes microcredenciales del programa, incluyendo módulos, lecciones, laboratorios y proyectos prácticos.

## Microcredenciales

1. Governance, Risk and Data Privacy  
2. Vulnerability Management  
3. System and Network Security  
4. Cloud Security 
5. Security Operations (SOC) 
6. Incident Response and Digital Forensics 

## Contenido del repositorio

Cada microcredencial incluye:

- módulos
- lecciones
- laboratorios
- documentación técnica
- proyectos

## Objetivo

Construir un portfolio técnico de ciberseguridad documentando el aprendizaje práctico mediante escenarios reales y ejercicios de seguridad.

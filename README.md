
# Cybersecurity Portfolio – IBM SkillsBuild

Este repositorio documenta mi aprendizaje práctico del certificado de ciberseguridad de IBM SkillsBuild.

Incluye documentación técnica, ejercicios, laboratorios y proyectos basados en escenarios reales.

## Áreas de ciberseguridad cubiertas

- Governance, Risk and Compliance (GRC)
- Vulnerability Management
- System and Network Security
- Cloud Security
- Security Operations (SOC)
- Incident Response and Digital Forensics

## Estructura del repositorio

learning-notes → documentación de aprendizaje 
labs → ejercicios prácticos 
docs → análisis y políticas de seguridad
projects → proyectos de seguridad

## Objetivo

Construir un portfolio técnico documentando habilidades prácticas de ciberseguridad.
